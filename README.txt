|Bubblemen VS: Championship Edition|
 ----------------------------------

HOW TO RUN
----------
Two executables are provided inside zip files. One for macOS users and one for Windows users. Simply copy the required zip file to your hard drive and open it. Execute the provided .app or .exe file respectively to run the game. The game should have no issues running on modern machines.

Here are the basic requirements needed by the Unity Player:
-OS: Windows XP SP2+, Mac OS X 10.8+, Ubuntu 12.04+, SteamOS+.
-Graphics card: DX9 (shader model 3.0) or DX11 with feature level 9.3 capabilities.
-CPU: SSE2 instruction set support.


HOW TO PLAY
-----------
The game is a 2D fighting game in the veins of Smash Bros. The goal of the game is to make your opponent's bubbleman pop by pushing it around the arena in a sumo-style deathmatch.

From the main menu, press enter and you will see the available game modes:
-Tournament Mode: Not available in this prototype.
-Multiplayer Mode: Play locally against one friend using the same keyboard.
-Online Mode: Play against one friend over a P2P network. (One friend must select "Host" and the other "Join")

After selecting your mode, select your favorite arena from the Arena Selection screen. In this prototype, only the Sink arena is available.

GAME CONTROLS
-------------
PLAYER 1
Move Left: A
Move Right: D
Jump: W
Attack: Left Shift

PLAYER 2
Move Left: < (Left Arrow Key)
Move Right: > (Right Arrow Key)
Jump: ^ (Up Arrow Key)
Attack: Right Shift

Attacking on the ground will punch, while attacking in midair will kick.

GOAL OF THE GAME
----------------
Try to make your opponent pop by pushing him against a wall. Every pop is worth 1 point, and the first player to gain 5 points wins.
